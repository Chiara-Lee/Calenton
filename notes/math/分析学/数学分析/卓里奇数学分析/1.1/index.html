1.1 几个常用不等式

Table of Contents

命题 1.3.1 (Bernoulli(伯努利)不等式) 设 h>-1,n\in \mathbb{N}_+ ，则有不等式成立

(1+h)^n\ge 1+nh

其中当 n>1 时等号成立的充分必要条件是 h=0

证： n=1 或者 h=0 时不等式显然成立（且均为等号成立），故以下仅讨论 n>1 和 h\ne 0 的情况。

将 (1+h)^n-1 作因式分解，

(1+h)^n-1=h[1+(1+h)+(1+h)^2+\cdots+(1+h)^{n-1}]\tag{1.1}

其中，上式的右侧即是级数

S=\sum_{k=0}^{n-1}(1+h)^k

根据等比数列求和公式，对于等比数列 \sum_{k=0}^{n-1}r^k ，其和为

S=\frac{1-r^n}{1-r}

代入可得

S=\sum_{k=0}^{n-1}(1+h)^k=\frac{(1+h)^n-1}{h}

故而（1.1）式成立。

等比数列求和公式：

对于等比数列

S_n=a+ar+ar^2+\cdots+ar^{n-1}

将两侧乘 r 得到

> rS_n=ar+ar^2+\cdots+ar^{n} >

两式相减得到

S_n-rS_n=a-ar^n

整理即得到

S_n=\frac{a(1-r^n)}{1-r}

当 r=1 时，等比数列即为 n 项 a 相加，因此结果即为 S_n=na

当 h > 0 时，在右边方括号内从第二项起都大于 1，因此就有：

(1 + h)^n - 1 > nh.

在 -1 < h < 0 时，在 (1.1) 右边方括号中从第二项起都小于 1，因此方括号中表达式之和小于 n 。由于 h < 0 ，因此又得到：

(1 + h)^n - 1 > nh.

为了应用的方便，可将 Bernoulli 不等式推广为双参数形式。

令 h = \frac{B}{A} ，其中 A > 0 ， A + B > 0 ，则条件 1 + h > 0 成立。将这个 h 代入 Bernoulli 不等式中，就可以得到下一个不等式。

命题 1.3.2 (Bernoulli 不等式双参数形式)
设有 A > 0 ， A + B > 0 ， n \in \mathbb{N}_+ ，则成立不等式：

(A + B)^n \geq A^n + n A^{n-1} B,

而且当 n > 1 时等号成立的充分必要条件是 B = 0 。

下面要介绍的是著名的算术平均值-几何平均值不等式，也简称为平均值不等式。它在两个实数的情况下包含了中学数学的三个基本不等式：

  1. \sqrt{ab} \leq \frac{1}{2}(a+b) \quad (a, b > 0),
  2. \frac{a}{b} + \frac{b}{a} \geq 2 \quad (a, b \text{ 同号}),
  3. a^2 + b^2 \geq 2ab \quad (a, b \in \mathbb{R}),

且仅当 a = b 时，以上三个不等式中等号成立。

命题 1.3.3 （算术平均值 - 几何平均值不等式）
设 a_1, a_2, \dots, a_n 是 n 个非负实数，则成立不等式

\frac{a_1 + a_2 + \cdots + a_n}{n} \geq \sqrt[n]{a_1 a_2 \cdots a_n},

其中等号成立的充分必要条件是 a_1 = a_2 = \cdots = a_n 。

证 1
一开始可以看出，如果在 a_1, a_2, \dots, a_n 中出现 0，则不等式已经成立。

又可以看出，这时等号成立的充分必要条件是其中每个数为 0。
因此以下只要对 a_1, a_2, \dots, a_n 为 n 个正数的情况来进行证明就够了。

应用数学归纳法。在 n=0 时结论已经成立。现设 n=k 时不等式已成立，然后讨论 n=k+1 。将 k+1 个正数 a_1, a_2, \dots, a_{k+1} 的算术平均值分解如下：

\frac{a_1 + a_2 + \cdots + a_{k+1}}{k+1} = \frac{a_1 + a_2 + \cdots + a_k}{k} + \frac{k a_{k+1} - (a_1 + a_2 + \cdots + a_k)}{k(k+1)}. \tag{1.2}

解释下上式的分解动机：首先我们来分析各项的含义：

  1. \frac{a_1 + a_2 + \cdots + a_{k+1}}{k+1} 代表 k+1 项的平均数。
  2. \frac{a_1 + a_2 + \cdots + a_{k}}{k} 代表前 k 项的平均数。

观察两平均数的差值，我们可以得到

\begin{align} a_{k+1}^{-}-a_{k}^{-}&=\frac{a_{1}+a_{2}+\cdots+a_{k}+a_{k+1}}{k+1}-\frac{a_{1}+a_{2}+\cdots+a_{k}}{k}\\ &=\frac{ka_{k+1}-(a_1+a_2+\cdots+a_k)}{k(k+1)} \end{align}

很显然，它就是（1.2）式中的右侧第二项。因此（1.2）式的分解动机就非常明显了，即： a_{k+1}^{-}= a_{k}^{-}+(a_{k+1}^{-}-a_{k}^{-})

然后将上述式右边的两项分别记为 A 和 B 。这时条件 A > 0 ， A + B > 0 满足，因此就可以应用 Bernoulli 不等式中双参形式作以下计算：

\left( \frac{a_1 + a_2 + \cdots + a_k + a_{k+1}}{k+1} \right)^{k+1} = (A + B)^{k+1} \geq A^{k+1} + (k+1) A^k B = A^k (A + (k+1) B) = \left( \frac{a_1 + a_2 + \cdots + a_k}{k} \right)^k \cdot a_{k+1} \geq a_1 a_2 \cdots a_k a_{k+1}.
