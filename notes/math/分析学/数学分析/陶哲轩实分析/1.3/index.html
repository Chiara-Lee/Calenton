2.3 乘法

Table of Contents

在上一节中，我们已经证明了所知道的所有关于加法和序的基本事实。为了节省篇幅以及避免赘述那些显然的事情，接下来我们将允许使用所熟悉的关于加法和序的代数法则，而不加进一步的说明。于是我们可以写出 a+b+c=c+b+a 这样的内容，不需要进一步解释说明。现在我们引入乘法。正如加法是重复的增量运算一样，乘法是重复的加法运算。

定义 2.3.1（自然数的乘法）

令 m 表示任意一个自然数，

我们定义 0\times m:= 0 来表示将 0 乘到 m 上。现在归纳假设我们已经定义了 n\times m （即把 n 乘到 m 上）。

那么，我们可以定义

(n++)\times m:= (n\times m)+m

以上，我们就定义好了自然数的乘法规则。

引理 2.3.2 （乘法是可交换的） 令 n 和 m 表示任意两个自然数，那么有 n\times m=m\times n 成立。

证明： 使用数学归纳法。

首先我们需要证明 m\times 0= 0 。当 m=0 时，我们有

0\times 0:= 0

归纳假设当 m=m 时 m\times 0= 0 是成立的，那么现在我们来证明 m=m++ 时等式成立。

\begin{align} (m++)\times 0&=(m\times 0)+0[(n++)\times m:= (n\times m)+m]\\ &=0 \end{align}

以上，我们就得到了基础情况 0\times m=m\times 0 。

接下来，我们需要证明

n\times (m++)=(n\times m)+n

当 n=0 时， 0\times (m++)=0=0\times m+0 （第一个等号来源于定义，第二个等号是为了交换形式而拼凑出来的）。假设当 n=n 时结论已经成立，即有 n\times (m++)=(n\times m)+n ，需证 (n++)\times (m++)=(n++)\times m+(n++)

\begin{align} (n++)\times (m++)&=[n\times(m++)]+(m++)\\ &=n\times m+n+(m++)\\ &=n\times m+m+(n++)\\ &=(n++)\times m+(n++) \end{align}

其中第三个等号是因为

\begin{align} &n+(m++)=n+(m+1)\\ &(n++)+m=(n+1)+m\\ &n+(m+1)=(n+1)+m\\ &n+(m++)=(n++)+m \end{align}

综上,引理得证。

我们将把 n\times m 简写为 nm ，而且按照惯例，乘法运算优先于加法运算。

引理 2.3.3（正自然数没有零因子） 令 n 和 m 表示任意两个自然数，那么 n\times m=0 当且仅当 n 和 m 中至少有一个为 0 .

证明： 设 n、m 均为正自然数，并且满足 n=c++、m=d++ ，其中 c、d 均为自然数。

\begin{aligned} & (n \text { is positive }) \wedge(m \text { is positive }) \\ \Rightarrow & \exists c, d \in N, c++=n, d++=m \\ \Rightarrow & n m=(c++) \times(d++)=c \times(d++)+(d++)=(c \times(d++)+d)++ \\ \Rightarrow & n m \text { is positive } \end{aligned}

命题 2.3.4（分配律）

对于任意自然数 a,b,c

\begin{align} &a(b+c)=ab+ac\\ &(b+c)a=ba+ca \end{align}

均成立

**证明：**由于乘法是可交换的，因此只需要证明第一个等式 a(b+c)=ab+ac 成立即可。

对 c 进行数学归纳。当 c=0 时，

\begin{align} &a(b+0)=ab\\ &ab+a0=ab\\ &a(b+0)=ab+a0 \end{align}

现在我们归纳性地假设 a(b+c)=ab+ac 成立，接下来要证明的是 a(b+(c++))=ab+a(c++) 。

\begin{align} &a(b+(c++))=a[(b+c)++]\\ &a[(b+c)++]=a(b+c)+a\\ &ab+a(c++)=ab+ac+a \end{align}

根据 a(b+c)=ab+ac ，因此可知 c++ 的情况亦成立。，那么到这里归纳过程就结束了。

命题 2.3.5（乘法是可结合的）

对于任意自然数 a,b,c

(a\times b)\times c=a\times(b\times c)

均成立。

证明：

对 b 进行数学归纳 : 对于 0: (a \times 0) \times c=0 \times c=0=a 0=a(0 \times c) 如果 b 是正确的，即 (a\times b)\times c=a\times(b\times c) , 那么

\begin{array}{l} (a(b++)) c=(a b+a) c=(a b) c+a c \\ =a(b c)+a c=a(b c+c)=a((b++) c) \end{array}

命题 2.3.6（乘法保持序不变）

如果 a,b 是满足 a 的自然数，并且 c 是正的，那么

ac

**证明：**因为 a ，因此存在某个正自然数 d 使得

b=a+d

对上式两端同时乘以 c ，然后利用分配律可得

bc=ac+dc

由于 c、d 都是正的，因此 dc 显然是正的，于是 ac

推论 2.3.7（消去律）

如果 a,b,c 均为自然数，且满足 ac=bc ，其中 c 不为零。那么

a=b

**证明：**利用序的三歧性（命题 2.2.13），我们有如下三种情况：

ab

若 a ，那么 ac 与题述条件相悖。 a>b 情况相同，因此只有 a=b 是对的。

命题 2.3.9（欧几里得算法） 设 n 是一个自然数， q 表示一个正自然数，那么就存在一个自然数 m 和 r 使得

0\le r n=mq+r

也就是说，我们可以用一个正自然数 q 去除以一个自然数 n ，从而得到商 m 和余数 r 。

证明：

固定 q ，对 n 进行数学归纳。 对 n=0 时，

0=0\times q+0 }

其中 m=0,r=0 。

假设当 n=n 时结论成立，也就是说，我们有 0\le r 和 n=mq+r ，那么

(n++)=(mq+r)++=mq+(r++) r

其中， r 是因为 r,q 都是自然数，因此，若 r 则 (r++) 不可能大于 q 。

若情况一成立，那么结论就成立。若情况二成立，那么

(n++)=mq+q=(m++)q=(m++)q+0

如此，结论成立。

正如我们可以用增量运算递归地定义加法运算以及用加法运算递归地定义乘法运算一样，我们也可以用乘法运算递归地定义指数运算。

定义 2.3.11（自然数的指数运算）
设 m 是一个自然数。我们定义 m^0:=1 来表示把 m 升到 0 次幂；特别地，定义 0^0:=1 。现在递归地假设对于某个自然数 n ， m^n 已经被定义了，那么我们定义

m^{n++}:=m^n\times m

例 2.3.12
于是譬如
x^1 = x^0 \times x = 1 \times x = x ，
x^2 = x^1 \times x = x \times x \ ，
x^3 = x^2 \times x = x \times x \times x ，
以此类推。根据归纳法可知，这种递归的定义把所有自然数 n 所对应的 x^n 都定义了。
